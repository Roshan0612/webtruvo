# Digital Growth Consultancy

A production-ready, corporate digital growth consultancy website built with Next.js App Router, TypeScript, Tailwind CSS, shadcn/ui, and Lucide icons.

## Highlights
- App Router with server components by default
- Strict TypeScript configuration
- SEO metadata and JSON-LD ProfessionalService schema
- Performance-focused UI with minimal client-side JavaScript

## Setup
1. Install dependencies with `npm install`.
2. Start the dev server with `npm run dev`.
3. Build for production with `npm run build`.

## Configuration Notes
- Update the canonical URL, email, and Calendly link to match your organization.
- Replace the hero SVG at public/images/consulting-hero.svg with a real professional image if desired.
